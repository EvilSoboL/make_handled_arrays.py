Инструкция к программе make_handled_arrays.exe
1. Поместить исходные данные с газоанализатора в папку modes, исходные данные должны выглядить следующим образом:

Tabs	Trel	O2(%)	H2(%)	CO(%)	CO2(%)	CH4(%)	SO2(ppm)	NO(ppm)	NO2(ppm)	N2O(ppm)	T(°C)	P()	Comments

24.12.2021 18:27:47	2	7.5	1.715	5.193	11.336	0.000	705	53	0	0	26.4	751.3	
...

2. В файле config.txt возможно изменение параметров работы алгоритма, если внесены изменения, перед символом ":" не должны присутсвовать символы в т. ч. пробел. Для вступления в силу изменений в алгоритм работы программы, необходимо закрыть config.txt, сохранив изменения.
3. Запуск программы осуществляется с помощью make_handled_arrays.exe
4. Результатом работы программы будут созданные папки с названием режима и окончанием _dir, например, 2021-12-24_S0D8CO2_dir. Количество созданных папок зависит от количества файлов с исходными данными.
5. Созданные папки содержат 3 txt файла, например:
 2021-12-24_S0D8CO2.txt, где первый столбец высота зонда газоанализатора, второй - время, с момента записи данных до вхождения в пламя в секундах, следующие столбцы - рассчитаное мат. ожидание последних 20 секунд нахождения зонда в пламение (config.txt - вторая строка).
